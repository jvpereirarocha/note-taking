
### Sorting Data

```
sort file -> Sort text in file
sort -k F -> Sort by key. F is the field number'
sort -r -> Sort in reverse order
sort -u -> Sort unique
```

The example below show a file named `example` that is unsorted. Using the sort command, we can see that the file content now is sorted
